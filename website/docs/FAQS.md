### What is BEAST MODE?
BEAST MODE is an AI-powered development platform that helps you understand and improve your code. It scans your code, answers questions, suggests improvements, and provides tools to help you build better code.

### Do I need to sign up?
No! You can use demo mode immediately. However, signing up gives you:
- ‚úÖ Saved scan history
- ‚úÖ Plugin installations
- ‚úÖ Team collaboration
- ‚úÖ Personal analytics

### Is BEAST MODE free?
Yes! We offer a generous free tier with full access to core features. Premium features are available with paid plans.

### How do I get started?
1. Go to the dashboard
2. Click "Quick Scan" in the Quality tab
3. Enter a GitHub repo (e.g., `facebook/react`)
4. See your quality score in 10 seconds!

## ‚ö° Quality Tab

### How long does a scan take?
- **Quick Scan:** ~10 seconds
- **Advanced Scan:** 30-60 seconds (depends on repo size)

### What does the quality score mean?
- **80-100:** Excellent (green) - Production ready
- **60-79:** Good (amber) - Minor improvements needed
- **Below 60:** Needs improvement (red) - Significant issues

### Can I scan private repositories?
Yes! Connect your GitHub account and grant access. We use secure OAuth authentication.

### What gets analyzed?
- Code quality and structure
- Security vulnerabilities
- Performance issues
- Best practices
- Test coverage
- Documentation

### How do I compare scans?
Click "Compare with Previous" to see:
- Score changes
- Issue improvements
- Trend analysis

### Can I export scan results?
Yes! Click "Export" to download results as CSV or JSON.

## üß† Intelligence Tab

### How does the AI assistant work?
It analyzes YOUR actual codebase (from recent scans) and provides specific answers, not generic advice.

### What can I ask?
Anything about your code! Examples:
- "What's the quality of my code?"
- "Suggest improvements for my project"
- "What are the biggest risks?"
- "How can I improve performance?"

### Do I need to scan first?
For best results, yes! The AI uses your scan data for context. Scan your repo, then ask questions.

### What are missions?
Missions are improvement goals you can track:
- Improve quality score to 90
- Add error handling to all routes
- Increase test coverage to 80%

### How accurate are recommendations?
Very accurate! Our AI has a 95%+ accuracy rate and learns from real-world usage.

## üì¶ Marketplace & Plugins

### Are plugins free?
Many plugins are free! Some premium plugins require a subscription. All free plugins are clearly marked.

### How do I know which plugins to install?
Check the Intelligence tab for personalized recommendations based on your project type and needs.

### How do I install a plugin?
1. Browse the Marketplace
2. Click "Install" on any plugin
3. Configure if needed
4. Start using immediately!

### Can I uninstall plugins?
Yes! Click "Uninstall" in the Installed Plugins section. Your configuration is saved for reinstall.

### How do plugin updates work?
Plugins automatically check for updates. You'll see a notification when updates are available. Click to update.

### Can I create my own plugins?
Yes! See the Plugin Development Guide in the Marketplace tab. We welcome community contributions!

### What if a plugin breaks my code?
You can instantly disable or uninstall plugins. All changes are reversible. We test all plugins before publishing.

## ‚ú® Improve Tab

### Is it safe to auto-fix code?
Yes! We recommend reviewing changes before committing. All fixes are reversible via git.

### What types of fixes are available?
- Remove console.log statements
- Add error handling
- Fix code quality issues
- Security improvements
- Performance optimizations

### Can I customize what gets fixed?
Yes! Configure fix preferences in Settings. You can enable/disable specific fix types.

### How do I undo a fix?
Use git to revert changes, or manually restore the original code. All fixes show a diff before applying.

### Does it commit automatically?
No! You choose when to commit. The fix is applied, then you review and commit manually.

## üêõ Troubleshooting

### Scan fails - what do I do?
1. Check repository URL format (`owner/repo`)
2. Ensure repo is public or access granted
3. Try again in a few moments
4. Check browser console for errors

### Plugin won't install
1. Check internet connection
2. Clear browser cache
3. Refresh the page
4. Try a different browser

### AI responses are slow
1. Check internet connection
2. Try a simpler question
3. Wait a moment and retry
4. Check if scans are recent

### Can't see my scans
1. Ensure you're signed in
2. Clear browser cache
3. Check browser console
4. Try a different browser

### Plugin not working
1. Ensure plugin is enabled
2. Check plugin configuration
3. Try reinstalling
4. Check plugin documentation
