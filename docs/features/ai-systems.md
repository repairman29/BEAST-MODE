## ðŸ’¡ Getting Started

### Enable All Systems
All systems are enabled by default when you initialize BEAST MODE:

```bash
beast-mode init
```

### Use Individual Systems
```bash
# Quality Engine
beast-mode quality check

# Code Roach (auto-fixes)
beast-mode quality check --fix

# Oracle AI (analysis)
beast-mode intelligence analyze

# Conversational AI (via dashboard)
beast-mode dashboard
```
