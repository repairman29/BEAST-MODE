## ðŸš€ Quick Start: New Project

### Create New Project

```bash
# Create project
mkdir my-vibe-project
cd my-vibe-project

# Initialize
npm init -y
npm install @beast-mode/core

# Initialize BEAST MODE
beast-mode init

# Start building!
```

### Add BEAST MODE

```javascript
// In your code
import { BeastMode } from '@beast-mode/core';

const beastMode = new BeastMode();
await beastMode.initialize();

// Use it!
const quality = await beastMode.getQualityScore();
console.log(`Vibe Score: ${quality.overall}/100`);
```
