## ðŸ“‹ Refinement Process

### Step 1: Generate Passes
```bash
python3 scripts/refine-visuals.py 10
```

### Step 2: Review Each Pass
- Open `assets/refinements/` folder
- Compare passes side-by-side
- Note improvements in each pass

### Step 3: Select Best Pass
- Identify best version for each visual
- Note what makes it best
- Document improvements needed

### Step 4: Update Source Files
- Adjust HTML/CSS based on best pass
- Fine-tune colors, spacing, effects
- Regenerate final version

### Step 5: Final Polish
- Generate final version
- Verify brand consistency
- Export to assets/
