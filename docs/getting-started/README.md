## Your Complete Guide to Getting Up and Running

**Welcome, vibe coder!** This guide will get you from zero to productive in just a few minutes.

## ðŸš€ Quick Start (3 Steps)

### Step 1: Install (30 seconds)

```bash
npm install -g @beast-mode/core
```

**Verify it worked:**
```bash
beast-mode --version
```

You should see: `1.0.0` âœ…

---

### Step 2: Initialize Your Project (1 minute)

Navigate to your project and initialize BEAST MODE:

```bash
cd /path/to/your/project
beast-mode init
```

This creates a `.beast-mode/` directory with your configuration. You're all set!

---

### Step 3: Check Your Code (2 minutes)

Run your first quality check:

```bash
beast-mode quality check
```

**What happens:**
- BEAST MODE scans your code
- Finds issues and opportunities
- Shows your quality score (0-100)
- Gives you recommendations

**That's it!** You're now using BEAST MODE. ðŸŽ‰
